# 🦆 Duck Farm Backend

This is the backend API for the **Duck Farm** app — a simple Express.js + TypeScript application that connects to a **MySQL database** hosted on **Google Cloud SQL**. It provides endpoints to manage users and their ducks.

---

## 🚀 Stack

- [Express.js](https://expressjs.com/) – Web server
- [TypeScript](https://www.typescriptlang.org/) – Type safety
- [Prisma](https://www.prisma.io/) – ORM for MySQL
- [MySQL](https://www.mysql.com/) – Relational database
- [Google Cloud SQL](https://cloud.google.com/sql) – Managed DB hosting

---

## 📦 Installation

```bash
git clone https://github.com/your-username/duck-farm-backend.git
cd duck-farm-backend
npm install
```
